üß≠ Quick Decision Guide: Which Pattern Should I Use?

User Need / Scenario Recommended Pattern Why?
Viewing details of an item Hierarchical Natural drill-down, easy to go back
Creating something new Modal Focused task, clear entry/exit points
Confirming a destructive action Alert Requires explicit acknowledgment
Choosing from multiple options Action Sheet Clear list of choices, easy to cancel
Showing contextual info about an element Popover Non-intrusive, points to source
Switching between major app sections Tab Bar Persistent access, parallel sections
Showing supplementary content (Maps-style) Resizable Sheet Flexible height, content stays visible
